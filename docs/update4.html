Development Update #4:
Entering Into a Smart Contract

Recap: March 15, 2018

This is the fourth in a series of developmental updates for Matcheos. We have now browsed through profiles, submitted offers, and received matches. In this post, we explore the user interface and process for entering into a smart contract.

Matcheos Smart Contract Demo:

Accepted Matches

Neither person has to accept a match. However, the first person to accept gets to select the type of smart contract.

Goals

Every contract has a specific goal. For example, you can interact with a person before deciding whether to become Facebook friends.

Specifics

When choosing a contract, you set the number of interaction steps and also how staked EOS is released back to each person.

Arbitration

A person can request arbitration for a breach in the Code of Conduct. This entails a small fee.

Work Smart Contracts:

Release All

Either person can end the contract at any point. When this occurs, all remaining EOS is released from the contract. However there is a 24 hour delay release to prevent arbitration frontrunning.

Networking

Work smart contracts can be used to help you find a job or business development opportunity.

Interviewing

Interaction steps can be used to assess the skills of a job candidate prior to a phone interview.

Education

You may want to reach out to someone about a particular work topic, if they are an expert.

Love Smart Contracts:

Love

Smart Contracts can have a goal where people meet-up out in the world. Geographic filtering can be used here.

Extensions

Matcheos will launch with a limited number of smart contracts. But the vision is to have hundreds or even thousands of available contracts. They could be as specific as a Picnic Smart Contract.

Developers

In the future, developers can create their own contracts (as a sort of Contract App Store) to be available as templates within each relevant section.

More Details:

That's it for this post. Next up will be a discussion about smart contract workflow. This will describe the process of interaction between the matched people as the contract proceeds. Stay tuned.
