Recap: May 3, 2018

Matcheos is happy to announce its first working Alpha release. After several weeks of development (see Updates #1-#5), the software is now ready for initial alpha testers and public code review. This alpha version only requires participants to provide an email address. Currently, the invitations are sent on a first-come-first-serve basis (...for those on the waiting list, please be patient). As the software gets hardened through testing/bug fixes, the rate of invitations will steadily increase. For now, the software should be considered as a 0.1 version. It is expected that the software is preliminary. Do not view it as a final release. There will be resets as the software upgrades to newer blockchain versions.

Matcheos' Mission

Matcheos wants to be a force for good. One of the core innovations of the Matcheos Smart Contract is to penalize scammers and trolls for violations of the Code of Conduct. The goal is to meet new people in a trustless and safe manner. Having said that, the crypto space is currently experiencing high-rates of scams. Please be vigilant and protect yourself. To use the Alpha version of Matcheos, you only need an email to receive the invitation. MATCHEOS DOES NOT ASK FOR PUBLIC OR PRIVATE EOS KEYS. You do not even need to have any EOS tokens to use the Alpha version. Any official Matcheos announcements will occur on multiple platforms - including this site. Please report any phishing scams or other malfeasance.
