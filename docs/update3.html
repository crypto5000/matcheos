Development Update #3:
First EOS Testnet Connection

Recap: March 11, 2018

This is the third in a series of developmental updates for Matcheos. The last post (see here) explained the matching algorithm. In this post, the user interface has been integrated with the EOS Testnet and various improvements have been made to the software.

Matcheos Demo:

New Features

Matcheos profiles now connect with the EOS Testnet, so EOS balances and offers reflect the running blockchain totals.

Love

Since the last update, Matcheos has renamed the "Dating" area to "Love" for aesthetics. It's now Love, Work, Friends.

Analytics

An analytics screen shows realtime offers both received and sent. Plus, you see how may times your profile has been viewed.

Settings

The settings screen allows you to update your picture and profile data at any time. Try different things to see what happens.

Signup:

New Features

Since Matcheos now connects with the EOS Testnet, the user interface for signup has been created.

EOS

To signup, you use your account username within the EOS network. This name is consistent across all EOS apps.

Login

You then login to Matcheos using your EOS account username and password.

Other

Matcheos will eventually support non-English languages. An age field is used to filter minors from adults.
